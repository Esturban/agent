# 144 — QLoRA Fine-Tuning

Fine-tunes a small LLM with 4-bit NF4 quantization (QLoRA) using `bitsandbytes`. Tracks VRAM before/after and shows what percentage of parameters are actually trainable. Runs on a free Colab T4 GPU.

**Run:** `python main.py` — requires a GPU; use Colab T4 for free tier
