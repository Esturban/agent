# 16-rag-eval-notebook

Hands-on evaluation workbook using **RAGAS** to score a RAG pipeline on Faithfulness, Answer Relevance, and Context Recall. Includes exercises that deliberately degrade the pipeline so you can see exactly how each metric reacts.

**Keys:** `OPENAI_API_KEY`
**Extra dep:** `ragas>=0.1.0` (already in `requirements.txt`)

```bash
# local notebook (recommended)
jupyter notebook examples/16-rag-eval-notebook/rag_eval_workbook.ipynb
```
