# 147 — Model Merging

Merges two HuggingFace model weight tensors using SLERP (spherical linear interpolation) and compares outputs from model A, model B, and the merged model side by side. No GPU needed — runs entirely on CPU.

**Run:** `python main.py` — no API key required; models downloaded from HuggingFace Hub
